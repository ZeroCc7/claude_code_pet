# V1.1 Core Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 建立 V1.1 的新存档、轻量背包、功德簿和首页入口，为青云山道与自动战斗提供稳定数据基础。

**Architecture:** 先在 Python 参考模型中定义并测试物品和 AI 任务记录，再同步到固件 `PetSaveData` 与 `GameState`。存档版本升级后只接受新结构，不迁移 V1.0；UI 第一阶段只调整页面枚举和首页导航，具体视觉页面在后续阶段实现。

**Tech Stack:** Python 3、pytest、Arduino C++、LittleFS A/B 双槽存档、ST7735S UI
