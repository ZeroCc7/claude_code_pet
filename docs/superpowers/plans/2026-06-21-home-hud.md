# 首页 HUD Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将首页顶部改为境界等级、本级经验和离线状态，底部改为双进度条、资源徽章和四键提示。

**Architecture:** 只修改 `GameUi` 首页绘制函数，不改变游戏状态和存档。顶部与底部使用独立绘制辅助函数；宠物动画继续只更新局部画布，不重绘 HUD。

**Tech Stack:** Arduino C++、Adafruit GFX、U8g2 中文字体、Python 源码结构测试
