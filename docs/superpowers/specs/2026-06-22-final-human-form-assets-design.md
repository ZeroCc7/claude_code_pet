# 最终人形形态素材设计

## 目标

为四种最终进化形态补齐正式像素美术，替换当前 `PetRenderer` 中的人形几何占位图：

- 太虚剑仙·凌霄：男性，青白剑道仙衣，墨玉麒麟角与青色剑域。
- 九转丹仙·瑶华：女性，朱白丹道仙衣，墨玉麒麟角与丹火莲华。
- 不灭武仙·镇岳：男性，玄金体修战衣，墨玉麒麟角与金雷拳罡。
- 万灵仙尊·清岚：女性，青碧御灵仙衣，墨玉麒麟角与御灵法环。

四者均以俊美人形为主体，保留明确的麒麟血脉特征，整体采用中国古风修仙像素画风。

## 素材结构

每种形态拆分为人物本体和独立特效两层：

1. 人物本体为四帧 `2×2` 待机动画。
2. 人物双脚使用统一底部锚点，四帧身高与比例保持一致。
3. 待机帧只包含呼吸、眨眼、发丝、衣摆和少量贴身仙气。
4. 完整技能特效为独立四帧 `2×2` 动画，不改变人物本体尺寸。
5. 原始生成图使用纯色 `#FF00FF` 背景，处理后输出透明 PNG。

固件中最终人物使用独立尺寸常量，不强制与现有 62×62 灵卵和幼年麒麟共用画布。目标人物帧尺寸不超过 `64×82`，并放入现有 `72×100` 宠物画布。

## 动画与触发

首页持续播放四帧人物待机动画，并保留低强度贴身仙气。

完整特效用于：

- K1 互动成功。
- AI 任务成功完成。
- 发生进化。

首页不播放完整技能特效，避免画面长期过亮。事件动画结束后恢复普通待机。

四种完整特效分别为：

- 太虚剑仙：青色剑域与短促剑气。
- 九转丹仙：金红丹火莲华。
- 不灭武仙：金色雷纹与拳罡。
- 万灵仙尊：青碧御灵法环与灵光。

## 固件接入

- 扩展宠物素材转换脚本，加入四种最终形态的人物帧。
- 为特效建立独立生成头文件和帧描述，不手工编辑生成产物。
- `PetRenderer` 根据 `PetForm` 选择对应人物帧。
- `GameUi` 继续通过离屏 `petCanvas_` 合成，再一次性提交屏幕，避免闪动。
- 事件特效只在首页宠物区域内局部刷新。
- 保留 `PREVIEW 3` 至 `PREVIEW 6` 检查四种最终形态。

## Flash 与性能

- 人物帧和特效统一转为 RGB565 与 1 位蒙版。
- 人物和特效分别存储，避免把大范围特效重复写入每个人物帧。
- 转换后编译固件，检查 Flash 和 RAM 未超限。
- 若四套完整特效导致 Flash 压力过高，优先缩减特效尺寸或颜色细节，不降低人物本体清晰度。

## 验证

1. 检查四套人物透明帧、脚底锚点、比例和轮廓一致性。
2. 生成动画预览，确认四帧循环无明显跳动。
3. 运行 Python 测试。
4. 编译固件并检查 Flash/RAM。
5. 烧录后通过 `PREVIEW 3` 至 `PREVIEW 6` 检查四形态。
6. 分别触发互动成功和 AI 完成，确认完整特效播放后能恢复待机。
7. 确认首页背景、人物和状态区域无闪动或覆盖。

## 本期范围

本期只补充四种最终形态的待机动画和事件特效，不制作攻击、行走、受伤或独立战斗动作。
