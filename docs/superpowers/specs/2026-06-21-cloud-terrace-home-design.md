# 云海仙台主界面设计

## 目标

替换当前偏暗且细节密集的洞府背景，让三种宠物在 128×160 屏幕上更大、更靠下，并与背景形成明显层次。

## 背景

- 中国古风修仙像素画。
- 主场景为高空云海仙台，远处可见淡青山峰、云瀑和柔和天光。
- 中央宠物区域保持简洁，不放置树枝、器物、文字或高对比纹理。
- 宠物身后使用偏亮、低细节的云海。
- 宠物脚下放置深青黑色悬浮仙台，带暗金边缘和少量青色符纹。
- 深色仙台用于承托浅色凌霄麒麟；明亮云海用于衬托深色镇岳麒麟。
- 顶部 18 像素状态栏和底部 48 像素数值区仍由固件覆盖。

## 宠物布局

- 宠物动态区域由 56×50 扩大为约 72×64。
- 正式宠物帧由 48×48 调整为 62×62。
- 宠物主体目标高度约 56 至 60 像素。
- 宠物区域整体下移约 10 像素，脚部落在悬浮仙台上方。
- 三种形态统一水平居中和底部锚点。
- 最终人形占位图保持可见，位置按新区域居中。

## 刷新

- 背景仍转换为 128×160、64 色、RGB565 固件资源。
- 宠物仍在离屏画布中与背景局部合成。
- 每个动画帧只更新新的宠物区域，不整屏重绘。
- 不允许出现黑框、拖影或闪烁。

## 验收

- 混沌灵卵、凌霄麒麟和镇岳麒麟明显大于上一版。
- 宠物位置不再偏上，视觉上落在仙台中央。
- 青白宠物和玄黑宠物都能与背景清楚区分。
- 状态栏和底部数值区不受背景调整影响。
- 编译后 Flash 和 RAM 仍有安全余量。
