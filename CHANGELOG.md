# 版本记录

本项目采用语义化版本号。

## v1.0.0 - 2026-06-23

首个完整可用版本。

### 设备与固件

- 完成 RP2040-Zero、ST7735S 彩屏、四按键和背光 PWM 支持。
- 完成培养、历练、首领战、重复挑战和奖励递减。
- 完成七种宠物形态、LV3 与 LV12 分支进化及最终人形麒麟动画。
- 完成六页面中文古风 UI、离屏合成和局部动画反馈。
- 完成 LittleFS A/B 双槽存档、CRC32 校验、版本迁移和任务去重。

### AI 工具联动

- 支持 Codex、Claude Code、OpenCode 和 CodeFree-O。
- 支持接引、推演、施法、炼器、等待、受阻、完成和空闲状态。
- AI 任务按持续时间结算经验、灵石和灵力。
- 同一来源与任务 ID 只结算一次。
- 提供 Hook 安装脚本、手动控制器和串口验收工具。

### 工程化

- 建立与固件规则对应的 Python 参考模型。
- 覆盖游戏规则、协议、Hook 载荷、UI 集成和串口验收测试。
- 提供 Arduino CLI 工具链、固件编译和烧录脚本。

### 验收

- 屏幕、按键、背光和 LittleFS 实机验收通过。
- AI Hook ACK、奖励、防重复及四种工具联调通过。
- 30 分钟诊断稳定性测试通过。
